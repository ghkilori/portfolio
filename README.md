# Портфолио лендинг

Готовый одностраничный сайт-портфолио в современном стиле с "призрачной" визуальной темой и плавными анимациями.

## Что внутри

- `index.html` - структура и контент сайта
- `styles.css` - стили, адаптив, эффекты и тема
- `script.js` - анимация появления секций + текущий год в футере

## Как запустить

1. Открой папку проекта в Cursor.
2. Запусти любой локальный сервер для статических файлов.
3. Открой `index.html` в браузере.

Пример с VS Code Live Server или аналогичным расширением.
